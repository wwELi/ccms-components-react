### 何时使用
页面局部处于等待异步数据或正在渲染过程时，合适的加载动效会有效缓解用户的焦虑。


### API
| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| size | loading大小，可选值：`default` `small`| string | default |
| type | loading样式（是否带背景层），可选值：`default` `layer`| string | default |
| tip | 自定义描述文案 | string | - |
| delay | 延迟显示加载效果的时间（防止闪烁） | number (毫秒) | - |
